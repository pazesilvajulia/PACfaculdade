Variáveis na Linguagem C

C é uma linguagem considerada de tipagem forte, ou seja, todas as variáveis precisam ter seus tipos definidos.

O que é tipo de variáveis?

Variáveis são locais onde são guardados dados, exemplo idade, altura e nome de uma pessoa tudo isso guardado num lugar chamado memória (RAM) do computador na, variáveis pode ser diversos tipos, os principais são os chamados de variáveis primitivas:

  • Float;
  • Int;
  • Char;
  • Bool;
  • String;

Tipos de variáveis:

  • Int (inteiro): É um valor numérico inteiro. Ex: int idade=18;
  • Float (ponto-flutuante): É um valor numérico que pode conter o ponto. Ex: float altura=1.8;
  • Char (caracter): É um tipo de variável que pode representar tanto como valores numéricos quanto valores que representam uma letra baseado na tabela ASCII.

Exemplo:

char primeiraLetraNome='F';

char idade =18;

Tabela ASCII:

String (texto): É um valor que representa um texto, uma lista/cadeia de letras. Diferente de outras linguagens, em C a string não é um tipo próprio, mas sim como uma lista de caracteres.

Exemplo:

char nome[12]= "Felipe Vieira";

Tabela com os tipos completos;

Definição de uma variável;

“tipo” “nome” = “valor”;

Tipos #Define e const:

#Define é uma diretriz da linguagem, usado para auxiliar na referência de dados textos por nomes, não sendo uma variável, mas uma indicação de um dado por um nome.

Exemplo:

#define PI 3.14159265359;

O valor do PI é constante e não muda;

Exemplo onde pode ser usado:

float areaCirculo= 3.14159265359 *2*radio; // O valor “3.14159265359” é o PI de forma implícita.

Usando define:

#define PI 3.14159265359;

float areaCirculo= PI*2*radio;

E o const?

Const cria uma variável imutável (não pode ser modificada após ser inicializada). A diferença em relação ao #define é que const cria uma variável na memória do computador, que pode ser usada para armazenar valores mais complexos, como listas (arrays) e estruturas de dados (struct).

Exemplo:

const int lista_numeros[3]= {5,3,99};

printf(“Primeiro número: %i”, lista_numeros[0]);

Regras de Nomenclatura de variáveis em C:

Linguagem não pode conter os Keyword (palavras-chave) que são palavras reservadas da própria linguagem.

Exemplo:

int 2soma = 25+50; // vai gerar um erro.

int soma2 = 25+50; // será aceito pela linguagem.

Não pode conter espaços no nome da variável:

float salario funcionarios; // irá causar um erro.

float salario_funcionarios; // está correto.
