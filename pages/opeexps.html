Operadores e Expressões

Em C, podem ser considerados operadores, símbolos que que nos permitem realizar operações matemáticas, relacionais, condicionais etc.

Operadores são símbolos que realizam operações em operandos.

Os operadores em C podem ser classificados em algumas categorias de acordo com sua funcionalidade:

Operadores Aritméticos

+ - / * %

Operadores Relacionais

== < > !=

Operadores Lógicos

&& || !

Operadores bit a bit
Bitwise

` & ^ ~

Operadores de Atribuição

= += -=

Em C existem 9 operadores aritméticos

Exemplo prático do uso dos operadores:

  • Código

  • #include <stdio.h>
  • // entrada e saída de dados (ex: printf)
  • #include <stdlib.h>
  • int main() {
    • int soma, subtracao, multiplicacao, divisao, modulo;
    • int a = 5;
    • int b = 2;
    • // Declarando a var "a" com o valor inteiro 5 e a "b" com valor inteiro 2
    • soma = a + b; // var "soma" recebe a + b
    • printf("%i", soma);
    • subtracao = a - b;
    • printf("%i", subtracao);
    • multiplicacao = a * b;
    • printf("%i", multiplicacao);
    • divisao = a / b;
    • printf("%i", divisao); // divisão por inteiros, resultado inteiro
    • modulo = a % b;
    • printf("%i", modulo);
    • a++;
    • printf("%i", a);
    • b++;
    • printf("%i", b);
    • a--;
    • printf("%i", a);
    • return 0; // Finaliza o programa com sucesso
  • }
