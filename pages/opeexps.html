Operadores e Expressões

Em C, podem ser considerados operadores, símbolos que que nos permitem realizar operações matemáticas, relacionais, condicionais etc.

Operadores são símbolos que realizam operações em operandos.

Os operadores em C podem ser classificados em algumas categorias de acordo com sua funcionalidade:

  • Operadores aritméticos;
  • Operadores relacionais;
  • Operadores lógicos;
  • Operadores bit a bit;
  • Operadores de atribuição;

Em C existem 9 operadores aritméticos:

  • Adição +: Soma dois operandos;
  • Subtração -: Subtrai dois operandos;
  • Multiplicação *: Multiplica dois operandos;
  • Divisão /: Divide dois operandos;
  • Modulo %: Retorno o resto de uma divisão;
  • Incrementação ++: Aumento o valor do operador por 1;
  • Decrementarão --: Diminui o valor do operador por 1;

Exemplo prático do uso dos operadores:


                #include <stdio.h>  
                #include <stdlib.h> 
                
                int main() {  
                    int a = 5; 
                    int b = 2; 
                
                    printf("a + b = %i\n", a+b); // Soma os dois valores 
                    printf("a - b = %i\n", a-b); // Subtrai b de a 
                    printf("a * b = %i\n", a*b); // Multiplica a vezes b 
                    printf("a / b = %i\n", a/b); // Divide a por b 
                    printf("a %% b = %i\n", a%b); // Retorna o resto da divisão de a/b 
                
                    // Esses operadores abaixo estão alterando o valor da variável original 
                    printf("a++ = %i\n", ++a); 
                    printf("a-- = %i\n", --a); 
                
                    return 0; 
                }
