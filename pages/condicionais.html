Estruturas Condicionais

1. Introdução às Estruturas Condicionais

As estruturas condicionais na linguagem C permitem que o programa tome decisões com base em determinadas condições. Elas são fundamentais para controlar o fluxo de execução do programa.

Nesta aula, vamos estudar as instruções if, else if, else e a estrutura switch-case.

2. Estrutura if, else if e else

A estrutura começa com a palavra-chave if, seguida de uma condição entre parênteses. Se essa condição for verdadeira, o bloco dentro das chaves será executado. Se não, outras condições podem ser testadas com else if, e se nenhuma for verdadeira, o bloco else será executado.

Sintaxe:

  • Código

  • if (condicao1) {
    • // executa se condicao1 for verdadeira
  • } else if (condicao2) {
    • // executa se condicao2 for verdadeira
  • } else {
    • // executa se nenhuma for verdadeira
  • }

Exemplo: Classificando faixa etária:

  • Código

  • int idade = 32;
  • if (idade < 18) {
    • printf("Você é menor de idade");
  • } else if (idade >= 18 && idade <= 64) {
    • printf("Você é adulto");
  • } else {
    • printf("Você é idoso");
  • }

3. Estrutura switch - case

A estrutura switch avalia uma variável e executa o código correspondente ao case com valor igual ao da variável. É ideal para comparações fixas.

Sintaxe:

  • Código

  • switch (expressao) {
    • case valor1:
      • // código executado
      • break;
    • case valor2:
      • // outro código
      • break;
    • default:
      • // executado se nenhum valor for correspondente
  • }

Exemplo: Exibindo o dia da semana:

  • Código

  • int diaDaSemana = 3;
  • switch (diaDaSemana) {
    • case 1:
      • printf("Domingo");
      • break;
    • case 2:
      • printf("Segunda-feira");
      • break;
    • case 3:
      • printf("Terça-feira");
      • break;
    • case 4:
      • printf("Quarta-feira");
      • break;
    • case 5:
      • printf("Quinta-feira");
      • break;
    • case 6:
      • printf("Sexta-feira");
      • break;
    • case 7:
      • printf("Sábado");
      • break;
    • default:
      • printf("Dia da semana inválido");
  • }

Se o valor da variável for 3, o programa imprimirá "Terça-feira".

4. Palavras-chave break e default

O comando break encerra a execução de um bloco switch ou um loop (for, while...). Ele evita que o código continue executando os próximos cases.

Já default é utilizado quando nenhum case anterior foi satisfeito.

  • Código

  • int day = 4;
  • switch (day) {
    • case 6:
      • printf("Hoje é sábado");
      • break;
    • case 7:
      • printf("Hoje é domingo");
      • break;
    • default:
      • printf("Ansioso pelo fim de semana");
  • }

5. Exemplo Completo com if

Este programa lê um número entre 1 e 7 e imprime o dia correspondente da semana. Se o número for inválido, exibe uma mensagem de erro:

  • Código

  • #include <stdio.h>
  • #include <stdlib.h>
  • int main(void) {
    • int valor;
    • printf("Digite um valor de 1 a 7: ");
    • scanf("%d", &valor);
    • if (valor == 1)
      • printf("Domingo\n");
    • else if (valor == 2)
      • printf("Segunda\n");
    • else if (valor == 3)
      • printf("Terça\n");
    • else if (valor == 4)
      • printf("Quarta\n");
    • else if (valor == 5)
      • printf("Quinta\n");
    • else if (valor == 6)
      • printf("Sexta\n");
    • else if (valor == 7)
      • printf("Sábado\n");
    • else
      • printf("Valor inválido!\n");
    • return 0;
  • }
